# semver-effect Documentation

Comprehensive documentation for `semver-effect`, a strict SemVer 2.0.0
implementation built on Effect.

## Installation

```bash
npm install semver-effect effect
```

For detailed installation options and your first program, see the
[Getting Started](./getting-started.md) guide.
